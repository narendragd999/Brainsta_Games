हँस-हँस के थक जाओ — मज़ेदार हिंदी गेम
Files at root:
- index.html
- style.css
- game.js
- README.txt

Instructions:
1. Unzip the folder on your computer or phone.
2. Upload these files to your phone's web server folder, or open index.html in a mobile browser.
3. The game is responsive and touch-friendly. Tap the play area or the "शुरू करें" button to start.
4. Game lasts 30 seconds. Tap characters to score.

Notes:
- All files are static and run in a browser (no backend required).
- Works offline if opened locally in many mobile browsers; for some Android WebViews you may need to serve via simple HTTP.
