This package contains a mobile-optimized HTML5 game with 10 levels.

How to use:
1. Unzip hashi_game.zip
2. Open index.html in a mobile browser or desktop browser (works in Chrome/Edge/Firefox).
3. Use pointer/touch to drag between islands to create bridges (horizontal or vertical only).
4. Max two bridges between same pair. Bridges cannot cross.
5. Level completes when all islands show the required number of bridges.

Files:
- index.html
- styles.css
- main.js
- levels.json

Notes:
Audio is generated using WebAudio (no external files).
If you want further polish (art assets, animations, build system), tell me and I can extend it.
